Project Structure

A visual guide to the folder and file organization of the Qt/QML Network Diagram Editor.

  • NetworkDiagramEditor/
    • src/ (C++ Source Files)
      • main.cpp
      • diagramcontroller.h
      • diagramcontroller.cpp
      • networkelement.h
      • networkelement.cpp
    • qml/ (QML User Interface Files)
      • main.qml
      • DiagramView.qml
      • NetworkItem.qml
      • Connection.qml
      • AddItemDialog.qml
      • qml.qrc
    • icons/ (SVG Icons)
      • server.svg
      • switch.svg
    • CMakeLists.txt (Build Script)
